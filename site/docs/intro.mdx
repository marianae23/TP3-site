# Introdução

## Sobre o projeto

O **SweetFlow** é uma aplicação desenvolvida com o objetivo de simular a gestão de uma doceria moderna, permitindo organizar produtos, encomendas e tarefas administrativas de forma estruturada e intuitiva.

Este projeto foi desenvolvido no âmbito académico com foco na aplicação de conceitos de programação, organização de sistemas e documentação técnica utilizando Docusaurus.

---

## Objetivos da aplicação

O sistema foi criado para auxiliar no controlo e gestão de uma doceria, oferecendo funcionalidades importantes para o funcionamento do negócio.

Entre os principais objetivos estão:

- Organização de produtos
- Gestão de encomendas
- Controlo de stock
- Gestão administrativa
- Simulação de ambiente empresarial
- Aplicação de boas práticas de desenvolvimento

---

## Funcionalidades principais

### Gestão de Produtos

O sistema permite organizar os produtos disponíveis na doceria, incluindo:

- Nome do produto
- Categoria
- Preço
- Quantidade em stock
- Descrição

---

### Gestão de Encomendas

As encomendas podem ser registadas e acompanhadas ao longo do processo.

O sistema permite:

- Criar encomendas
- Atualizar estados
- Consultar pedidos
- Organizar informações dos clientes

---

### Área Administrativa

A aplicação inclui ferramentas administrativas para facilitar a gestão do negócio.

Algumas funcionalidades incluem:

- Atualização de produtos
- Gestão de stock
- Controlo de encomendas
- Organização do catálogo

---
