# Manual do Administrador

## Introdução

A área administrativa do SweetFlow foi desenvolvida para permitir o controlo e organização das operações principais da doceria.

O administrador possui acesso a funcionalidades relacionadas com produtos, encomendas, promoções e gestão interna do sistema.

---

# Gestão de Produtos

O administrador pode controlar os produtos disponíveis na doceria.

## Funcionalidades disponíveis

- Adicionar produtos
- Editar produtos
- Remover produtos
- Atualizar preços
- Gerir categorias
- Gerir stock

---

## Informações dos Produtos

Cada produto pode conter:

| Informação | Descrição |
|---|---|
| Nome | Nome do produto |
| Categoria | Tipo de doce |
| Preço | Valor do produto |
| Stock | Quantidade disponível |
| Sabor | Sabor do produto |
| Cor | Cor principal |
| Tamanho | Tamanho disponível |

---

# Gestão de Stock

O sistema permite controlar a disponibilidade dos produtos.

## Funcionalidades disponíveis

- Atualizar quantidades
- Consultar stock disponível
- Identificar produtos indisponíveis
- Organizar lista de compras

---

# Gestão de Encomendas

O administrador pode acompanhar todas as encomendas realizadas pelos clientes.

## Funcionalidades disponíveis

- Consultar encomendas
- Atualizar estados
- Consultar histórico
- Gerir pedidos

---

## Estados das Encomendas

As encomendas podem possuir diferentes estados.

| Estado | Descrição |
|---|---|
| Pendente | Pedido recebido |
| Em preparação | Produto em produção |
| Finalizada | Pedido concluído |
| Entregue | Produto entregue ao cliente |

---

# Gestão de Promoções

O sistema inclui ferramentas para gestão de promoções.

## Funcionalidades disponíveis

- Criar promoções
- Atualizar descontos
- Gerir campanhas
- Organizar produtos promocionais

---

# Gestão de Datas Bloqueadas

O administrador pode definir datas indisponíveis para encomendas.

## Exemplos

- Feriados
- Datas de manutenção
- Períodos sem produção

---

# Histórico de Compras

O sistema permite consultar o histórico de encomendas realizadas.

## Informações disponíveis

- Produtos comprados
- Datas
- Valores
- Estados das encomendas

---

# Objetivos da Área Administrativa

A área administrativa foi desenvolvida com foco em:

- Organização
- Controlo operacional
- Facilidade de gestão
- Estruturação de dados
- Simulação de ambiente empresarial

---

# Considerações Finais

O SweetFlow procura oferecer ferramentas simples e organizadas para auxiliar na administração e funcionamento da doceria.
