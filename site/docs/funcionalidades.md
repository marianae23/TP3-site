# Funcionalidades

## Visão Geral

O SweetFlow foi desenvolvido para auxiliar na gestão de uma doceria, oferecendo funcionalidades tanto para clientes como para administradores.

O sistema permite organizar produtos, gerir encomendas e melhorar o controlo das operações da doceria.

---

# Área do Cliente

A área do cliente permite consultar produtos e efetuar encomendas de forma simples e intuitiva.

---

## Visualização de Produtos

Os clientes podem consultar os produtos disponíveis na doceria.

### Funcionalidades disponíveis

- Visualizar produtos
- Consultar detalhes
- Consultar avaliações
- Procurar produtos por nome

---

## Sistema de Filtros

O sistema inclui diferentes filtros para facilitar a procura de produtos.

### Filtros disponíveis

| Filtro | Objetivo |
|---|---|
| Categoria | Organizar tipos de produtos |
| Sabor | Procurar sabores específicos |
| Tamanho | Filtrar tamanhos disponíveis |
| Cor | Filtrar produtos por cor |

---

## Gestão de Encomendas

Os clientes podem efetuar e acompanhar encomendas diretamente no sistema.

### Funcionalidades disponíveis

- Criar encomendas
- Cancelar encomendas
- Consultar estado da encomenda
- Consultar tempo estimado

---

## Avaliações

O sistema permite interação entre clientes através de avaliações.

### Funcionalidades disponíveis

- Avaliar produtos
- Consultar avaliações de outros clientes

---

# Área do Administrador

A área administrativa oferece ferramentas para gestão completa da doceria.

---

## Gestão de Produtos

O administrador pode controlar os produtos disponíveis no sistema.

### Funcionalidades disponíveis

- Gerir produtos
- Adicionar produtos
- Remover produtos
- Alterar preços
- Gerir stock

---

## Gestão de Promoções

O sistema inclui funcionalidades relacionadas com promoções.

### Funcionalidades disponíveis

- Criar promoções
- Atualizar promoções
- Gerir descontos

---

## Gestão de Encomendas

O administrador pode acompanhar e atualizar as encomendas realizadas.

### Funcionalidades disponíveis

- Alterar estado de encomendas
- Consultar checkout
- Consultar histórico de compras

---

## Gestão Organizacional

O sistema também inclui ferramentas de organização interna.

### Funcionalidades disponíveis

- Gerir datas bloqueadas
- Gerir lista de compras

---

# Objetivos do Sistema

O SweetFlow foi desenvolvido com foco em:

- Organização de produtos
- Gestão de encomendas
- Simulação de um sistema empresarial
- Aplicação de conceitos de programação
- Utilização de Git e GitHub
- Criação de documentação técnica
