# Funcionalidades

## Homepage
A homepage apresenta informações principais sobre a doceria, promoções e navegação para as restantes páginas.

![Homepage](/img/homepage.png)


## Catálogo
O catálogo permite visualizar os produtos disponíveis organizados por categorias.

## Filtros
Os utilizadores podem filtrar produtos por:
- categoria
- preço
- sabor
- promoções

## Encomendas
A plataforma permite realizar encomendas de produtos de forma simples e organizada.

## Administração
A área de administração permite gerir:
- produtos
- stock
- promoções
- encomendas

## Promoções
Os produtos em promoção são destacados automaticamente na plataforma.
